# Студия-детейлинг SOVA

Статический сайт для студии-детейлинг SOVA с информацией об услугах, галереей работ и контактами.

## Дизайн

- **Цветовая схема:** Черно-серый и бронзовый
- **Стиль:** Премиум, минималистичный
- **Адаптивность:** Полностью адаптивный дизайн

## Структура проекта

```
frontend/
├── index.html          # Главная страница
├── about.html          # О нас
├── services.html       # Услуги и цены
├── gallery.html        # Галерея работ
├── contacts.html       # Контакты
├── css/                # Стили
│   ├── styles.css
│   └── components/
│       ├── header.css
│       └── footer.css
└── images/             # Изображения
```

## Запуск

Просто откройте www.s0va.site в браузере.


## Страницы сайта

- **Главная** (`index.html`) - главная страница с обзором услуг
- **Услуги** (`services.html`) - прайс-лист всех услуг
- **Галерея** (`gallery.html`) - примеры выполненных работ с фильтрацией
- **О нас** (`about.html`) - информация о компании
- **Контакты** (`contacts.html`) - контактная информация

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript (только для фильтрации галереи)
